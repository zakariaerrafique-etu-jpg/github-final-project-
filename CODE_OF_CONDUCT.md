# Code de conduite

## Notre engagement

Nous nous engageons à créer un environnement ouvert et accueillant pour tous.

## Nos normes

Les comportements attendus incluent le respect, la collaboration et la communication constructive.

## Responsabilités d'application

Les mainteneurs du projet sont responsables de l’application du code de conduite.

## Portée

Ce code de conduite s’applique à tous les espaces du projet.

## Application

Les violations seront examinées et traitées de manière appropriée.

## Directives d'application

Les actions peuvent inclure un avertissement, une suspension ou une exclusion.

## Attribution

Basé sur le Contributor Covenant, version 2.0.
